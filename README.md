# موتوشاب — پروتوتایپ نسخه ادغام‌شده (Motoshub × Mizito)

این مخزن یک **پروتوتایپ کلیک‌پذیر (clickable prototype)** از رابط کاربری جدید پلتفرم «موتوشاب» است؛ حاصل ادغام مفهومی هسته‌ی اجتماعی **Motoshub** (مبتنی بر Oxwall/PHP) با قابلیت‌های مدیریت پروژه/دانش الهام‌گرفته از **Mizito** و نیازمندی‌های قرارداد «استقرار و توسعه شبکه اجتماعی دانشمند» در هر سه فاز.

توجه: این یک محصول نهایی نیست — تمام داده‌ها mock هستند و هدف، اعتبارسنجی UX/UI و معماری اطلاعات پیش از شروع توسعه‌ی واقعی بک‌اند است. طراحی بدون استفاده از ایموجی و صرفاً با آیکون (lucide-react) انجام شده تا زبان بصری کاملاً رسمی/سازمانی باشد.

## دسترسی آنلاین

```
https://cheedafar.ir/
```

## اجرای محلی

```bash
npm install
npm run dev
```
