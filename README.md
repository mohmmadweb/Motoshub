# موتوشاب — پروتوتایپ نسخه ادغام‌شده (Motoshub × Mizito × Mattermost)

این مخزن یک **پروتوتایپ کلیک‌پذیر (clickable prototype)** از رابط کاربری جدید پلتفرم «موتوشاب» است؛ حاصل ادغام مفهومی هسته‌ی اجتماعی **Motoshub** (مبتنی بر Oxwall/PHP)، قابلیت‌های مدیریت پروژه/دانش الهام‌گرفته از **Mizito**، و ارتباطات درون‌سازمانی در سطح **Mattermost** (کانال‌های عمومی/خصوصی، رشته‌ی پاسخ، وضعیت حضور چهارگانه، reaction، پیام سنجاق‌شده/ذخیره‌شده، دستور اسلش، وب‌هوک/بات، حساب مهمان) — همه پشت یک هویت و یک تجربه‌ی کاربری واحد، مطابق نیازمندی‌های قرارداد «استقرار و توسعه شبکه اجتماعی دانشمند» در هر سه فاز.

توجه: این یک محصول نهایی نیست — تمام داده‌ها mock هستند و هدف، اعتبارسنجی UX/UI و معماری اطلاعات پیش از شروع توسعه‌ی واقعی بک‌اند است. طراحی بدون استفاده از ایموجی و صرفاً با آیکون (lucide-react) انجام شده تا زبان بصری کاملاً رسمی/سازمانی باشد.

قابلیت دیگر: کلید میانبر **Ctrl/⌘+K** در هر صفحه‌ای از اپ، یک Quick Switcher (مشابه Mattermost) باز می‌کند که می‌توان با آن مستقیم به هر صفحه، کانال یا کاربر رفت.
