# موتوشاب — پروتوتایپ نسخه ادغام‌شده (Motoshub × Mizito)

این مخزن یک **پروتوتایپ کلیک‌پذیر (clickable prototype)** از رابط کاربری جدید پلتفرم «موتوشاب» است؛ حاصل ادغام مفهومی هسته‌ی اجتماعی **Motoshub** (مبتنی بر Oxwall/PHP) با قابلیت‌های مدیریت پروژه/دانش الهام‌گرفته از **Mizito** و نیازمندی‌های قرارداد «استقرار و توسعه شبکه اجتماعی دانشمند».

⚠️ این یک محصول نهایی نیست — تمام داده‌ها mock هستند و هدف، اعتبارسنجی UX/UI و معماری اطلاعات پیش از شروع توسعه‌ی واقعی بک‌اند است.

## دسترسی آنلاین

این پروژه روی روت دامنه‌ی اختصاصی زیر منتشر می‌شود (نه روی `mohmmadweb.github.io`، تا با رزومه‌ی شخصی قاطی نشود):

```
https://cheedafar.ir/
```

برای فعال‌شدن این دامنه، یک رکورد DNS در Cloudflare و تنظیم Custom domain در GitHub Pages لازم است — جزئیات در پاسخ نهایی گفتگو آمده.

## اجرای محلی

```bash
npm install
npm run dev
```

## ساختار صفحات

| مسیر | توضیح |
|---|---|
| `/` | لندینگ معرفی پلتفرم |
| `/login` | ورود (نام‌کاربری/رمز یا OTP) |
| `/app/dashboard` | فید/داشبورد اصلی |
| `/app/groups` , `/app/groups/:id` | گروه‌های تعاملی |
| `/app/forum` | انجمن |
| `/app/chat` | گفتگوی متنی |
| `/app/knowledge` | مدیریت دانش / آرشیو اسناد و قراردادها |
| `/app/projects` , `/app/projects/:id` | مدیریت پروژه (بورد + گانت) — هسته Mizito |
| `/app/profile/:id` | پروفایل کاربر/سازمانی |
| `/app/admin` | پنل راهبری چندسازمانی (Multi-tenant) و فعال/غیرفعال‌سازی ماژول‌ها |
| `/app/notifications` | اعلان‌ها |
